# Joinery System 2.0

Joinery 2.0 upgrades Grainline from isolated rule edits to connected construction intent.

## Highlights

- Connection graph (`PartConnection`) for part-to-part reasoning.
- Dependency ordering to avoid invalid cascading adjustments.
- Preset application service for fast, repeatable construction patterns.
- Impact preview in UI for before/after dimensional clarity.

## Current presets

- Frame and panel case
- Dovetailed drawer
- Dado drawer
- Mitered case

## How to use

1. Open `/labs` joinery panel.
2. Pick a preset or single rule.
3. Apply and review impact preview deltas.
4. Validate output in Materials and print packet.

## Safety note

Joinery calculations are engineering defaults, not absolute shop truth. Confirm every critical clearance with your hardware/jointing workflow before cutting.
