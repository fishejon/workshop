# Furniture Types

Phase 10 ships multi-type casework support through shared configuration + generation services.

## Dresser

- Graduated drawer-first layout.
- Recessed base and optional overhang.
- Optimized for bedroom storage workflow.

## Console

- Shallow depth profile.
- Top drawer with open shelf zone.
- Optional face frame and larger top overhang.

## Bookshelf

- Tall shelf-first layout.
- Adjustable shelves via feature flags.
- Optional base storage opening.
