# Hardware Integration

Grainline generates a hardware schedule from cut-list semantics to reduce missed purchasing.

## What is generated now

- Drawer slide recommendation (undermount default, full extension).
- Drawer pull count aligned to detected drawer rows.

## Where it appears

- `/print` under **Hardware schedule**
- CSV export as a second section (`hardware_type,quantity,specs,notes`)

## Recommendation discipline

- Treat hardware output as a planning baseline.
- Verify manufacturer clearances, mounting, and load limits before final order.
