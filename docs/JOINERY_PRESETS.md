# Joinery Preset Catalog

## Frame and Panel Case

- Rails to stiles with mortise-and-tenon.
- Panel/back capture in grooves.
- Best for shaker-style carcass zones.

## Dovetailed Drawer

- Dovetail side/front/back logic.
- Grooved drawer bottom capture.
- Best for premium drawer boxes.

## Dado Drawer (Simple)

- Front/back captured in side dados.
- Grooved bottom support.
- Best for fast, robust utility drawers.

## Mitered Case

- Miter edge reinforcement using rabbet-like spline channel assumptions.
- Best for clean-edge case forms requiring hidden corner treatment.
