# Purchase Intelligence

Phase 11 upgrades the Materials workflow from totals-only to decision support.

## What is new

- Strategy engine generating four scenario lenses.
- Side-by-side comparison table for board count, waste, transport, complexity, and optional cost.
- Stock type selector (`surfaced` vs `rough`) with explicit guidance.
- Pricing input modal with typical range references.

## Where users see it

- `Materials` tab in `BuyListPanel`.
- Works with the same project parts data as the yard list/cut layout.

## Planning guidance only

Scenario, waste, and cost values are engineering estimates:

- Verify board availability and stock width at the yard.
- Confirm transport constraints against real vehicle setup.
- Confirm pricing directly with suppliers.
